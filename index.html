鱼叨 | 社会现象有感

发表于 2023-10-10 更新于 2023-11-04 分类于 认知革命

一件被遗忘的事，上周看到一篇新闻，大桥上男子打双闪停车，自缚双腿，跳河自杀。看完视频心猛地一揪，临死前还打双闪，这样温柔的人，在自杀时却如此毫不犹豫，且不给自己留活下来的希望。然而在这样的一篇新闻以及关于类似的自杀事件讨论中，仍会有很多的人去谴责男子自私，不去考虑自己的家里人，说男子脆弱，种种言论。人们在面对已死之人时尚且如此求全责备，而在面对抑郁、自残、求死的人时，更是苛责，未经他人之苦的妄下定论，如同一双双魔爪将人推入深渊。

近年来我愈发的感觉到和长辈们之间的代沟之深，或许是受教育程度，或许是疫情导致的大环境变革，青年一代产生了一定程度的认知觉醒，不再接受企业的PUA，大批年轻人在毕业后选择考研考编考公，人口负增长，丁克/不婚不育的思想在我认识的一些优秀的00后女生里并不少见，其实我挺开心的，说了那么久的男女平等，终于是亲眼所见。

上周我恰好和当老师的姑姑聊及这个话题，当我问到她如何看待丁克/不婚不育的这些人时，她不加犹豫的评判出“自私”二字。我微微一愣，这个结果其实并不出我所料，但我以为她至少会思考一番，能够更加抽丝剥茧的去看待这个问题。在我看来，这是个时代问题，很多年前我最早听到丁克的概念时，只有很少一部分高知的群体选择这种方式，而如今这种观念已经蔓延到了大量的20上下的青年群体中，除却高学历高收入人群，更多的普通人也加入其中。疫情应当是造成这种现象的直接原因吧，整个大环境一下子布上阴霾，人人自危，我们更多的开始关心自身的健康，珍惜鲜活的生命。

鱼叨 | 我怎么是这种人

发表于 2023-09-09 更新于 2023-11-04 分类于 认知革命

暑假之前我还深陷在上一段恋情的失败中无法走出，即便已经分手了大半年，我仍像一个BT一样试图去挽回什么。最终了了，开始摆烂去过自己的躺平生活。

暑假因为摄影认识了新的朋友，crush，一下子好像看到一道光，但是此时的我已经自我厌恶极深，不敢去触碰，遥遥相望已经觉得满足。于是乎又不想摆烂了，想去变得更好一些，我不希望自己在未来的某些时刻，遇到让自己心动的女生或者优秀的人想去结交时，连走上前开口，大大方方与之交流的勇气都没有。

回到标题，我怎么是这种人呢？既恋爱脑又渣男的感觉，不过这样也好，至少证明不是非她不可，这个世界上有几十亿人，我也会有一次又一次的心动，那个与我同频的人，终将会在某个时刻相遇，而我所需要做的，就是让自己不断变得更好。

《也许你该找个人聊聊》读书笔记

发表于 2022-06-29 分类于 半日闲

读后感

最近的情绪乱糟糟的，看书也似囫囵吞枣，《也许你该找个人聊聊》，从书名就直接吸引了我，翻开书页后，读到 “ 假如不痛苦，你就不曾体味真实的人生；假如你也深陷痛苦，你凭什么帮助别人？”，我便知道，就是这本书了。

作者是专业的心理医生，书的内容围绕作者本人的心理治疗和其患者的心理治疗展开，四个来访者：认为身边所有人都是蠢货的好莱坞纸片人；刚新婚就身患绝症的大学女教师；年近七十与子女不合觉得虚度此生的老太太；爱情频频受挫的年轻女孩；以及失恋后崩溃的治疗师自己。从不同的抑郁患者身上，我总是能看到自己的影子，正如书中所言“痛苦并没有高低之分”，我便总能从这些截然不同的境遇中看到一些相似的东西，起到一些共鸣。

《地下室手记》读书笔记

发表于 2022-05-08 更新于 2022-06-01 分类于 半日闲

读后感

重抑确诊，三年来一直有着极其强烈的精神内耗，从陀翁笔下的无名主人公身上，仿佛看到了放大版的自己，直击心脏的共鸣。
原生家庭和社会带给我的影响，人性复杂的内心世界，我越是试图去解开弄懂这一切，越是陷入无尽的痛苦之中，我原以为这样的痛苦会永远伴随我直到死去，感谢遇到陀翁的手记，让我看到自身的遭遇并非芸芸众生之中的绝一例，虽这世上或许没有人能够感同身受，但有文字给我以力量，已是此生一最大的幸事，这些年也越来越能够从书籍之中获取心灵的慰藉，内心得以逐渐趋于平静。
我浅薄的阅历并不能使我读懂此书所要阐述的所有深刻哲理，虽在代入主人公的思考里感受到那么多那么多的共鸣，但我仍未能搞懂这些复杂，扭曲，拧巴的思想斗争以及与之矛盾鲜明反差的行为举止，更加准确的说，我不知道自己该如何去与之相处，是的，我与之是何其的相似，以至于在结书之后，这问题已自动被我带回现实之中，而现实中的我，应是还做不到也早早躲进地下室，叹气。
需要感谢后记的总结，带着我一点点理清了一些事情，身份焦虑和身份认同，或许这就是我一直在苦苦挣扎的原因，茫茫寻找的答案，或许我需要用一生去寻觅解决之法，但不会后悔，胡乱的感慨就到这吧。
最后，突然想起王小波的这句——“我活在世上,无非想要明白些道理,遇见些有趣的事。”
