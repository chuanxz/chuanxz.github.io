ArrayList 和 LinkedList 区别

List 是 Java 的一个接口，常见的实现类有**ArrayList**和**LinkedList**。

  • 底层：前者的底层数据结构是数组，后者是链表
  • 添加删除：ArrayList插入删除受元素位置影响，LinkedList是链表不受影响。
  • 查找：ArrayList支持高效随机访问，LinkedList不支持。

发表于 2022-04-13 更新于 2022-06-02 分类于 编程学习 ， 计算机基础

模型

OSI 七层：物理层、链路层、网络层、传输层、会话层、表示层、应用层
TCP/IP 四层：网络接口层、网际层、运输层、应用层
五层协议：物理层、链路层、网络层、运输层、应用层
每层协议：

  • 物理层：RJ45、CLOCK、IEEE802.3（中继器、集线器、网关）
  • 链路层：PPP、FR、HDLC、VLAN、MAC（网桥、交换机）
  • 网络层：IP、ICMP、ARP、RARP、OSPF、IPX、RIP、IGRP（路由器）
  • 传输层：TCP、UDP、SPX
  • 表示层：JPEG、MPEG、ASII
  • 应用层：FTP、DNS、Telnet、SMTP、HTTP、WWW、NFS
