《牧羊少年奇幻之旅》读书笔记

发表于 2021-11-24 更新于 2022-06-01 分类于 半日闲

读后感

忘了第一时间写这篇书评，因为看完这本书的那会处于比较迷茫的时间段，且我未能及时领悟保罗在此书中所传授的“炼金术”，书里书外，皆云里雾里。今日得闲，便想起补上这篇书评来了。
我在书中学到的第一个道理为—“万事皆为一物”，事物存在着语言之外的表达方式，当我们在学习一些新的事物的时候，会发现自己以前曾体验过，反之我们可以把学习到的东西用在以后学习其他的新事物上，若是掌握了这种脱离于语言的表达方式，就可以解读整个世界了。
另一个道理是—“倾听你的心声。心了解所有事物，因为心来自世界之魂，并且总有一天会返回那里”。当我们跟随心的声音，就可以寻找到自己的宝藏所在之处。我们不能让自己的心沉默，佯装听不到它的呐喊。我们需要去感知它对于生活和世界的看法，增加对自己的了解，只有这样，才能不会遭受意外的打击。
第三个道理—“夜色之浓，莫过于黎明前的黑暗”，这句话的意思也很简单，就是风雨之后见彩虹呗，多了程度上的描述，在我们经历最坎坷艰难的时候，一定要坚信，黎明的曙光即将在前方照亮。
该书讲述了许多道理，这些道理都可谓是人生的“炼金之术”，今日只能记起这些，之后二刷再补充。

《人生海海》读书笔记

发表于 2021-11-24 更新于 2022-06-01 分类于 半日闲

读后感

最近我的人生又开始变得奇奇怪怪，重新下载了王者荣耀，并浪费了三天本应该学习的时光，回想月初的第一周做过的少有的有意义的事情，就是阅读了《人生海海》这一本书，感谢这使我在今夜能有所记录。
合上书页的那一刻，我脑海中第一时间闪过的是另一本同样令我回味良久的名作——《百年孤独》，两本书同样描述的是围绕着小镇的家族兴衰史，虽是相隔大洋彼岸的两个国度，背景风格迥异，但所描绘的那种宏大的磅礴之势，却如出一辙。带给我的关于岁月长河的思考，也有着许多的相似之处。爷爷聪明一世，却因为犯下的一个小错误害的整个家族走向了死亡，家道中落，孩子们逃亡纷纷逃离小镇，有的甚至远渡重洋，上校风光一世，最后却退化的像一个孩子，完全靠着妻子的照顾度日。从内战到抗日再到文革，小镇居民的人生发生了翻天覆地的变化，有逐渐受到打压而落寞了的，也有借机发横财走向富裕的，也有直接被枪毙死去的，总结为一个词，我认为就是—“戏剧化”，正应了书名所描述的那样，人生海海，有突然潮起潮落。到了该书的末尾，我才知道作者笔下的人生海海是什么，“人生海海，敢死不叫勇气，活着才叫勇气。你要替我记住这句话，我要不选到它，死几回都不够”。这是一句闽南话，其真实含义是指人生应像大海一样宽广，教人好好活而不是去死，如醍醐灌顶。

今夜的思路特别混乱，其实在阅读完全书之后，我就感觉到有很多东西需要去消化，这是一本我必须二刷回顾的好书，甚至让我想再去复习一遍《百年孤独》。夜色已浓，先记录这部分感想，待得闲时再补充完全。

《基督山伯爵》读书笔记

发表于 2021-11-24 更新于 2022-06-01 分类于 半日闲

读后感

感谢自己在过去的半个月能够静下心来看完基督山伯爵这本长篇幅的巨著。
主角埃德蒙·唐泰斯那传奇般的人生经历，使我意识到自己在生活中所遇到的那些挫折，跌入谷底的反差，与之相比是如此的不值一提。若非这是一本世界名著，其中的故事之离奇只会使我觉得过于扯淡，于现世毫无参考价值。可如今我不得不去深度的思考该书在大仲马精妙的文笔之下，所欲传达的深刻含义。
回忆主角如何以一己之力推翻了三个已经声名显赫的家族，莫尔塞夫家族被揭露战争时期的恶行，背叛上级导致家族破灭，妻子儿子离家出走后自杀身亡。维尔福家族的继母为自己的小儿子谋取家产而将一家老小相继毒死，法官维尔福逼迫继母和她的儿子自己服毒赎罪，同一天，在判顶贝尔代托的谋杀罪时，被指认自己是他的亲生父亲，又引出一段陈案。银行家唐格拉尔原先一直想把自己的女儿嫁给卡瓦尔坎蒂亲王，但是被揭露之后，其女儿和同伴出逃。他自己也逃到意大利取款，后被大盗帕克劫去所有财富。故事的结局，瓦朗蒂娜和莫雷尔重新相聚，伯爵带着海黛远走高飞，仍不失为一个美好的结局。
全文篇幅巨长，大仲马对每一个人物的刻画都相当之细致，每一个家族的由盛到衰，细细品之过后，便能感受到其清晰严谨的故事情节。文中传达的财富观，是非观，人生价值观，都引我深思… …

《姜子牙》观后感

发表于 2021-11-24 更新于 2022-06-09 分类于 半日闲

这是 20 年看的最后一部电影，同时也是 21 年写的第一篇影评，即是终章，亦是序曲。
动漫讲述了封神大战之后，姜子牙因放走狐妖被贬北海界后的事情，在追寻狐妖真相的道路上，太公的执着令我动容，在与元始天尊的对话中，我陷入了长久的深思，舍一人而救苍生，这样的斩善之举，是世间真善么？我不由得联想到著名的电车难题，作为道德难题，它并没有客观正确的解法，动漫中折射的道理已经上升到我不能理解的哲学高度，我开始在网络上搜寻相关的讨论和影评。看到一段保罗在《黑客与画家》中的话，“如果自己就是潮水中的一部分，怎么能看见潮流的方向呢？你只能永远保持质疑”。
故事渐进，我才意识到电车难题只是动漫所要讲述的一小片，后续的问题演变成了讨论神明对于人间善恶是否应该拥有决定权，神明对世间的统治，需要斩断的是心中的善，“于一人恶，于天下善”是师尊的评判标准，而太公则认为人间善恶应由自身决定，无需神明介入。这个问题的难度丝毫不低于上一个，而对于这个新问题，我无法作出自己的见解，经历不够，阅历不足。
《姜子牙》所要阐述的道理，在格局上比《哪吒》要高很多，也正因此在网上的评价要低一些，影片的张力不够去描述如此宏大的哲理，观影者的阅历不足以去领会这样深刻的立意。我为自己在这样一部优秀的作品之前所表现出来的无知表示惭愧，亦希望自己在新的一年里能够继续欣赏到更多这样优质的影片和书籍。
以新年心愿作结：在新的一年里，多看，多想，多写。
