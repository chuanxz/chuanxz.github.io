《想见你》观后感

发表于 2021-11-24 更新于 2022-06-01 分类于 半日闲

在被疯狂安利之下，我终于忍不住打开了这部豆瓣高分网剧，后遂无法自拔，一口气看完了 26 集，怎么形容观后的心情呢，一个词——跪感十足。不管是双向穿越的悬疑情节，缤纷的摄影画面，余音绕梁的插曲，演员们的完美演技，都令我深深折服。剧中每一个鲜明的人物形象都让我印象深刻：黄雨萱/陈韵如：女主一人分饰两角，竟能表现出如此鲜明的差别，黄的阳光外向，独立干练（小声 bb：女神！！），陈的文静内向，抑郁。你说你能演出这两种感觉就很厉害了，还能演出陈扮演黄的样子是什么神仙操作！李子维：一句话，许光汉少年感十足！莫俊杰：男二真是太太太温柔细心惹～陈思源：可爱叛逆的弟弟，有趣！谢芝齐：有点心理扭曲…但是我觉得他很可怜，莫名的感触。And，这神仙导演是哪里来的灵感，这种双向穿越，穿穿穿的剧情都想的出来？最后还帮我把时间线给捋顺了？？结论：我果然应该再多读点书！不然以后连偶像剧都看不懂了！！！
对了，提一下剧中最令我心颤的情节：男主李子维在经过多次穿越之后，发现无论自己如何奋力地想要去改变过去，都无济于事，岁月侵蚀，想要屈服于命运时，却发现年少的自己竟发生了一点微小的变化，再加之后来女主的努力，竟一点点打破了这个时空的闭环！太感人惹！但是！这是什么道理呢？我是否，也可以凭借一点点的努力，不甘，不认命，去克服我的劣根性，和人性的弱点对抗，实现自己的改变，阶级的上升呢？不管怎样，都要持续努力不放弃吧，奥利给！

《大话西游》观后感

发表于 2021-11-24 更新于 2022-06-01 分类于 半日闲

依稀记得儿时看月光宝盒那会，什么都看不懂，只感到荒谬和笑意。而今时隔多年后重温这部星爷和朱茵的经典，一口气看完了月光宝盒和大圣娶亲，满满的情绪从心底溢出。什么是爱情，又什么是无奈，至尊宝拿着月光宝盒一次又一次想要回到过去，挽救心爱的白晶晶，最后却误回五百年前，遇到了紫霞仙子。故事的发展相当曲折，记得后来那段至尊宝和菩提的对话，昨晚你念晶晶的名字 98 次，哦，晶晶是我的娘子，还有个叫紫霞的你念了 784 次，啊，这紫霞一定欠你很多钱。先是笑出了声，而后陷入了沉默。你骗过了所有人甚至妄图骗过自己，却发现仍骗不过自己的内心，拔出紫青宝剑的至尊宝，和紫霞仙子的孽缘，明明是上天安排的最大嘛～而最后两人却并未在一起。紫霞被迫与牛魔王成亲，为了挽救紫霞，至尊宝不得不戴上紧箍咒化身齐天大圣。戴上紧箍，就必须斩断情丝，而放下紧箍却无法拯救自己的心上人。这该是怎样的一种无奈呢，至尊宝企图以恶人的姿态结束和紫霞的恋情，人世间最痛苦的事情想必莫过于此。最后在快撞上太阳之际，师徒四人再度使用月光宝盒回到过去，至尊宝看到自己和紫霞的化身立于墙头，义无反顾的帮助他们紧紧相拥，最后转身踏上了西天的征途。曲终，满满的苦涩。紫霞无谓的坚持，至尊宝的七百次梦呓，齐天大圣狗一样的转身离去，令人唏嘘。意中人是盖世英雄，即使踏着七色云彩归来，终究还是错过了。我想至尊宝真的放下了紫霞么，不然，不戴紧箍就救不了你，戴上紧箍就不能爱你，这样的无奈，在我看来就像是生活中千千万万个现实面前无奈的缩影。而我们终将面对现实，需要作出决定向前迈步。齐天大圣用他的行为告诉了我们他的选择，一个盖世英雄该做出的抉择。在大话里有太多个性鲜明的人物令我印象深刻，也有太多经典的话语值得我反复咀嚼，夜已深，文思很乱，先做这片初稿，之后若有想到，继续修改补充。——致永远的经典

一段都没分，看来这篇是我孩童时期写的…

清醒 | 一个人是如何废掉的？

发表于 2021-11-17 更新于 2022-05-30 分类于 认知革命

作为一个这些年被抑郁/躁郁毁掉的人，时常反刍自身的种种，其实早在很久之前在知乎看到过这个回答，然而然而，还是不可避免的走向堕落…我自问过无数次自己是如何废掉的，却仍于事无补，借这次机会将这个问题记录下来，聊以宽慰。

一、长期封闭环境

这个环境不单单指像牢笼一样困在一个物理环境里，同样包括心灵的环境，与外界社会的人际沟通。这同样是我在考研那年陷入抑郁的主要原因，当时虽人在校园，却已同身边的同学老师都几近断绝了联系，每天独来独往。作家李尚龙曾经说过：“在大城市里，搞废一个人的方式特别简单。给定一个安静狭小的空间，给你一根网线，最好再加一个外卖电话。好了，你开始废了”
