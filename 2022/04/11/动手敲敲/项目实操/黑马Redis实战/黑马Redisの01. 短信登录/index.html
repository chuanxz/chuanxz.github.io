集群的 session 共享问题

问题描述：多台 Tomcat 不共享 session 存储空间，导致切换 tomcat 服务后数据丢失。

session 替代方法应满足：

  • 数据共享
  • 内存存储
  • key、value 结构

答案-> Redis

Redis 替代 session 的业务流程

因为并发时 Redis 是共享的，
保存验证码时需要以 phone 作为 key。
保存用户时，以随机 token 作为 key。
